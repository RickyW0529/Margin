# 0203 持仓基础面板 — 实施计划

## 1. 任务目标

实现当前持仓面板基础总览与单个持仓基础详情视图：总资产、可用现金、持仓市值、今日/累计盈亏、组合波动率、最大回撤、行业暴露、风格暴露、高风险持仓数、即将发生的重要事件。单持仓详情在本任务内交付成本与盈亏、原始买入逻辑、操作历史；当前证据、估值变化、模型排名变化、催化剂时间线、失效条件由 06/08/09 后续模块填充，本任务预留字段与空态。

## 2. 工作项拆解

- 0203.1 持仓总览视图 — 总资产/现金/市值/盈亏/波动率/回撤/暴露/高风险数/事件。
- 0203.2 单持仓基础详情视图 — 成本盈亏/买入逻辑/操作历史，预留证据/估值/排名/催化剂/失效条件空态。
- 0203.3 持仓 API 对接 — GET /portfolios/{id}、/positions、/risk、/positions/{id}/thesis。
- 0203.4 前端 Next.js 面板页面 — 持仓总览与详情页。

## 3. 依赖关系

- 前置：0202（成本与风险数据）。
- 被依赖：0901（投资逻辑状态展示）、08（持仓复核视图）。
- 外部依赖：前端框架 Next.js。
