# 0801 候选卡片与首页 — 实施计划

## 1. 任务目标

实现研究候选面板首页信息层级（市场状态摘要、今日候选、现有持仓复核、高优先级风险、拒绝判断、策略运行状态）与候选卡片（含股票/价格/排名/状态/估值区间/安全边际/价值陷阱风险评分/事件窗口/催化剂/反方理由/证据数量等级/观察条件/失效条件/策略版本/「不是买卖指令」提示）。以可审计的 run/item 模型组织查询。

## 2. 工作项拆解

- 0801.1 Research Run Query Service 与 Dashboard BFF — run/item 查询接口。
- 0801.2 首页信息层级 — 六类信息聚合。
- 0801.3 候选卡片 — 全字段展示，含「不是买卖指令」提示。
- 0801.4 前端 Next.js 面板页面 — 首页与候选列表。
