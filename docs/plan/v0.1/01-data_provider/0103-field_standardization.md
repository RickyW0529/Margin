# 0103 字段标准化 — 实施计划

## 1. 任务目标

将外部数据经字段映射、代码映射、单位和币种统一、时间标准化后产出标准数据事件，供存储层 ODS→DWD→PIT 分层消费。统一 A 股代码格式、复权单位、财报币种与时间口径。

## 2. 工作项拆解

- 0103.1 字段映射与代码映射 — 外部字段→标准字段、股票代码统一（如 000001.SZ 口径）。
- 0103.2 单位与币种统一 — 复权因子、金额单位、财报币种。
- 0103.3 时间标准化 — event_at/published_at/available_at 口径对齐。
- 0103.4 标准数据事件发布 — 输出至存储层 DWD，记录映射规则版本。

## 3. 依赖关系

- 前置：0102（AKShare/Tushare 接入产出原始数据）。
- 被依赖：0104（质量校验基于标准化结果）。
- 外部依赖：无。

## 4. 工时估算

- 0103.1：2 天
- 0103.2：2 天
- 0103.3：2 天
- 0103.4：1 天
- 合计：7 天。

## 5. 里程碑与交付物

- M1：字段/代码映射规则集可用（第 2 天）。
- M2：单位币种与时间口径统一（第 6 天）。
- M3：标准数据事件发布至 DWD（第 7 天）。

## 6. 验收动作

- 同一股票在 AKShare/Tushare 两个源经标准化后字段一致；
- 映射规则版本化记录，变更可追溯；
- 对应 spec 01 §2 / §4。
